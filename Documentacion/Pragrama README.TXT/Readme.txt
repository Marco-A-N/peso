*** Cálculo de Índice de Masa Corporal (IMC) ***
=========================================================

Descripción:
------------
Este programa permite calcular el Índice de Masa Corporal (IMC) ajustado según la edad y el sexo. Además, categoriza el IMC en diferentes niveles y permite guardar los datos en un archivo CSV.

Requisitos del Sistema:
-----------------------
- Sistema Operativo: Windows, macOS, o Linux
- Python 3.6 o superior
- Librerías: 
  - tkinter
  - PIL (Python Imaging Library)
  - csv
  - os

Instalación:
------------
1. Asegúrate de tener Python instalado en su sistema.
2. Instale las librerías necesarias.
